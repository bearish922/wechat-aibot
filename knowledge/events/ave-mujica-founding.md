# Ave Mujica的成立

## 起点：祥子的坠落

丰川祥子退出CRYCHIC后经历了：母亲瑞穗病逝→父亲被丰川家族驱逐→祥子选择跟随父亲→靠打工维持生计→从月之森退学转入羽丘。在MyGO!!!!!演奏《春日影》时，祥子在观众席崩溃逃离，随后打电话给幼年好友三角初华——「让我忘记一切」。

## 组建过程

祥子以制作人身份策划Ave Mujica「假面舞会」企划：
1. 首先联系三角初华（当时已为人气偶像组合sumimi成员），以一句「拜托了」说服初华加入
2. 通过初华认识了足迹遍布30个乐队的专业贝斯手八幡海铃
3. 若叶睦作为CRYCHIC旧部，以「祥子要坏掉了」为由决定加入——带着赎罪心理
4. 最后招募美女YouTuber祐天寺若麦（喵梦），祥子看中的是她的影响者话题性而非鼓技
5. 五人佩戴面具使用拉丁语代号，以「Ave Mujica」之名主流出道→武道馆

## 各代号含义

- **Oblivionis（忘却）**：祥子，想忘却家族枷锁和伤痛
- **Mortis（死亡）**：睦，需要让依赖祥子的自己死去重生
- **Doloris（悲伤）**：初华，卸下偶像伪装直面悲伤
- **Timoris（恐惧）**：海铃，最怕无法被信任
- **Amoris（爱）**：喵梦，学习付出真情实意

## 与MyGO的关系

Ave Mujica与MyGO!!!!!是CRYCHIC解散后分裂出的「双子乐队」。祥子是两团的核心纽带——她伤害了MyGO的灯、素世和立希，同时又是Ave Mujica的创建者。两条故事线在Ave Mujica动画中交汇，CRYCHIC五人在第七集完成和解与告别仪式。
